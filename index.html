01. About Me

Hi, my name is Evgenii. I'm not just a tester—I'm a QA architect who builds testing strategies, tears down fragile code, and helps teams deliver products that don't embarrass them. I've been in the trenches of software quality for years, and I know every trick in the book.

I've mastered the art of breaking things (intentionally). Whether it's functional, API, performance, or UI/UX testing—I attack from every angle. Jira, Confluence, TestRail, Qase.io, Youtrack—these are my playgrounds. I wield Postman, Newman, curl, and DevTools like a ninja, and I've got SQL queries tattooed on my brain. Playwright? Cypress? I can automate faster than devs can blink.

My GitHub repos are packed with testing frameworks, utilities, and tools that make QA life easier. Check them out—there's everything from custom bug hunters to API validators to automation scaffolds. I don't just test; I build the future of testing.

REST APIs fear my scrutiny. Databases bend to my SQL. And if your product isn't battle-tested, it will be after I'm done with it.

Easy Sniff

Browser Extension for Real-Time Network Monitoring

Chrome extension for testers that intercepts all HTTP(S) requests, displays them in a convenient interface, shows methods (GET, POST, PUT, DELETE), status codes, response times, and even works with WebSocket messages. Perfect for API testing, debugging, and analyzing network activity of your application.

  • Real-time request monitoring
  • Filtering by URL, method, status, type
  • WebSocket protocol support
  • Sorting and quick search
